# Sistem Jadwal Kuliah

Aplikasi praktikum React Native + Expo SDK 54 (React Native 0.81, React 19).
Docs SDK: https://docs.expo.dev/versions/v54.0.0/

Catatan scope:
- Data HARDCODE di `data/scheduleData.ts` (tanpa backend / AsyncStorage / CRUD / form input).
- Styling memakai StyleSheet.create + token tema di `constants/theme.ts` (tanpa library UI eksternal).
